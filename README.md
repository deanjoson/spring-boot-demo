# spring-boot-demo
基于SpringBoot的各种使用demo

## 项目简介
Spring Boot Demo 是结合日常工作中使用到的技术按单一原则进行拆分按由简到繁的步骤逐步升入提供，便于后续进行系统回顾SpringBoot的使用，也可作为SpringBoot入门的参考信息

## 开发环境
- JDK 1.8
- Maven 3
- IntelliJ IDEA ULTIMATE（Lombok）
